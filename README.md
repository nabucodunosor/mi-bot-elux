# Elux Bot — Telegram

## Variables de entorno (configurar en Railway)
- `TELEGRAM_TOKEN` — el token que te dio BotFather
- `ANTHROPIC_API_KEY` — tu API key de Anthropic

## Archivos necesarios
- `bot.py` — código principal
- `productos.json` — catálogo de productos (subir junto con el código)
- `requirements.txt` — dependencias
- `Procfile` — instrucción para Railway
